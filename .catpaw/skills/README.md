# 项目专属 Skills

本目录存放针对本项目沉淀的 Claude 技能配置，用于快速执行常见任务。

## 目录结构

```
.catpaw/skills/
├── README.md           # 本文件
└── [.skill-name]/      # 各技能子目录
```

## 使用方法

在 Claude 对话中，使用 `/skills` 命令或直接描述任务，Claude 会自动匹配相关技能。

## 技能开发指南

每个技能可以是：
- **单文件**: 一个 `.md` 描述技能行为
- **多文件**: 包含示例代码、测试用例的完整包

技能应包含：
1. `description.md` - 技能描述与触发条件
2. `action.sh` 或 `action.ts` - 执行的脚本（可选）
3. `examples/` - 使用示例（可选）
