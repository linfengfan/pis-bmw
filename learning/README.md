# 学习成果目录

本目录是我在 AI 前端工程化实践中沉淀的核心模式，作为定期复习的活体文档。

## 核心协作模式

| 文件 | 主题 | 触发场景 |
|------|------|----------|
| `规划前置原则.md` | 规划前置原则 | 多文件/跨模块变更 |
| `原子化变更原则.md` | 原子化变更原则 | 任何代码变更 |
| `规范即文档原则.md` | 规范即文档原则 | 规范与代码同步 |
| `Skills系统.md` | Skills 系统 | SOP 复用 |
| `Bridge-Rule模式.md` | Bridge Rule 模式 | 跨系统上下文桥接 |
| `三级降级策略.md` | 三级降级策略 | 辅助功能容错 |
| `人类角色重新定义.md` | 人类角色重新定义 | 整体流程对齐 |

## 复习原则
- 每次大型变更后，更新对应文件
- 按需加载，不要一次性引入所有模式
- 模式是"指路"，不是"复制"
